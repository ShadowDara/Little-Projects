# open-wt-on-Hotkey
same as open CMD on Hotkey but with Mingw64, because the ps terminal supports multiline Input

### Hotkey
```
win + shift + b
```

[More Infos](https://github.com/ShadowDara/open-cmd-on-Hotkey)

## Warning
the executable will may be shown as false positive, if you dont trust me, you can install
AutoHotkey for your System and compile the script ahk script yourself!
